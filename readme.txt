== Description ==
Comming soon plugin allows you to quickly  launch under construction page for your website. Collect E-mails and connect with users on Social Networks.

= Features =

* Works with All Themes
* Social connectivity(Facebook, Twitter, Google+)
* Email Capture

= Requirements =
This plugin requires PHP 5 or PHP 7 and WordPress 3.0 or greater. It works in standard WordPress environments. It is strongly recommended to use WordPress in the newest version.

== Installation ==

1. Go to Plugins > Add New. 
2. Searh "Comming soon simple"
3. install and activate the WordPress Plugin.
